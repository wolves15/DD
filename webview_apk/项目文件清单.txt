# WebView APK 项目文件清单

本项目包含以下文件：

## 核心文件
- main.py - 应用主代码
- buildozer.spec - Buildozer打包配置
- requirements.txt - Python依赖

## 构建文件
- .github/workflows/build-apk.yml - GitHub Actions自动构建配置
- .gitignore - Git忽略文件
- Dockerfile - Docker构建配置
- build_apk.sh - Linux打包脚本

## 文档
- README.md - 项目说明文档
- GITHUB_BUILD_GUIDE.md - GitHub构建详细指南

## 测试文件
- test.html - HTML功能测试页面
- test.bat - Windows测试脚本
- test_simple.py - 简化版测试脚本
- test_connection.py - 连接测试脚本
- test_connection.bat - 网络测试脚本

## 工具文件
- package_project.bat - Windows打包脚本
- package_project.ps1 - PowerShell打包脚本
- build_guide.bat - 构建操作指南

## 使用方法

### 方法1: GitHub Actions（推荐）
1. 将所有文件上传到 GitHub 仓库
2. GitHub Actions 会自动构建 APK
3. 在 Actions 页面下载 APK

### 方法2: 本地构建
1. 安装 WSL (Windows Subsystem for Linux)
2. 运行 build_apk.sh 脚本
3. APK 输出到 bin/ 目录

详细说明请查看 README.md 和 GITHUB_BUILD_GUIDE.md
